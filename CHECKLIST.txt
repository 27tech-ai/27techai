PHASE 1: GET YOUR API KEYS (15 minutes)
================================================================================

[ ] 1. Get GitHub Personal Access Token
       → Go to: GitHub → Settings → Developer Settings
       → Personal Access Tokens → Tokens (classic)
       → Generate new token
       → Select: repo, workflow
       → Copy the token (ghp_xxxxx)

[ ] 2. Get OpenRouter API Key (Recommended)
       → Go to: https://openrouter.ai
       → Sign up (free)
       → Get API key (sk-or-v1-xxxxx)
       → Has free credits!

[ ] 3. (Optional) Get OpenAI API Key
       → Go to: https://platform.openai.com
       → Sign up
       → Get API key (sk-proj-xxxxx)

[ ] 4. Get Google AI API Key (Optional)
       → Go to: https://ai.google.dev
       → Sign up
       → Get API key

PHASE 2: SETUP ON YOUR COMPUTER (10 minutes)
================================================================================

[ ] 1. Open PowerShell
       → cd C:\Users\Hossi\Desktop\QODER\dashboard

[ ] 2. Install Python packages
       → pip install -r requirements.txt

[ ] 3. Generate encryption key
       → python generate_key.py
       → 📝 COPY the key shown!

[ ] 4. Edit encrypt_secrets.py
       → Open: dashboard/encrypt_secrets.py
       → Line 5: Replace YOUR_GENERATED_KEY_HERE
       → Fill in your API keys:
         • GITHUB_USERNAME = "your-username"
         • GITHUB_REPO = "your-repo-name"
         • GITHUB_TOKEN = "ghp_xxxxx"
         • OPENROUTER_API_KEY = "sk-or-v1-xxxxx"

[ ] 5. Encrypt your secrets
       → python encrypt_secrets.py
       → ✅ Creates: secrets.enc

[ ] 6. Update dashboard.py

       → Use same key from step 3

[ ] 7. Test dashboard
       → python dashboard.py
       → ✅ Should open with 4 tabs
       → Check ⚙️ Settings tab - API keys should show ✅

PHASE 4: UPLOAD TO GITHUB (5 minutes)
================================================================================

[ ] 1. Open PowerShell in project folder
       → cd C:\Users\Hossi\Desktop\QODER

[ ] 2. Initialize Git
       → git init

[ ] 3. Add files
       → git add .
       → (secrets.enc automatically blocked!)

[ ] 4. Commit
       → git commit -m "27TechAI - Initial setup"

[ ] 5. Set branch name
       → git branch -M main

[ ] 6. Connect to GitHub
       → git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git
       → (Replace with YOUR URL)

[ ] 7. Upload!
       → git push -u origin main
       → May ask for GitHub login
       → Use your username and token as password

❓ TROUBLESHOOTING:
================================================================================

Problem: "git: command not found"
→ Install Git: https://git-scm.com/download/win

Problem: "Permission denied" on push
→ Use token as password, not GitHub password

Problem: Site not updating
→ Wait 2 minutes, then hard refresh (Ctrl+F5)

Problem: Dashboard won't open
→ pip install -r requirements.txt

Problem: API errors
→ Check API keys are correct in encrypt_secrets.py
