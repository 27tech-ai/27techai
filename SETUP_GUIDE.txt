3. STEP-BY-STEP SETUP
================================================================================

STEP 1: Install Python Dependencies
-------------------------------------
Open terminal in the dashboard folder and run:

cd dashboard
pip install -r requirements.txt

This will install:
- cryptography (for encryption)
- requests (for API calls)
- tkinter (usually comes with Python)


STEP 2: Generate Encryption Key
-------------------------------------
Run the key generator:

python generate_key.py

You'll see output like:
🔑 Your Encryption Key:
xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

⚠️ COPY THIS KEY - You'll need it in the next step!


STEP 3: Configure Your Secrets
-------------------------------------
1. Open: dashboard/encrypt_secrets.py
2. Replace YOUR_GENERATED_KEY_HERE with your actual key
3. Fill in ALL your API keys:

secrets = {
    # GitHub Configuration
    "GITHUB_USERNAME": "your-username",
    "GITHUB_REPO": "your-repo-name",
    "GITHUB_TOKEN": "ghp_xxxxxxxxxxxx",
    
    # AI API Keys (add at least one)
    "OPENROUTER_API_KEY": "sk-or-v1-xxxxx",
    "OPENAI_API_KEY": "sk-proj-xxxxx",
    "ANTHROPIC_API_KEY": "sk-ant-xxxxx",
    "GOOGLE_AI_API_KEY": "xxxxx",
    "HUGGINGFACE_API_KEY": "hf_xxxxx",
    
    # Optional: Notifications
    "TELEGRAM_BOT_TOKEN": "xxxxx",
    "DISCORD_WEBHOOK_URL": "https://discord.com/api/webhooks/xxxxx",
    
    # Site Configuration
    "SITE_URL": "https://yourusername.github.io/yourrepo",
    "ADMIN_PASSWORD": "your-secure-password"
}


STEP 4: Encrypt Your Secrets
-------------------------------------
Run the encryption script:

python encrypt_secrets.py

This creates: dashboard/secrets.enc
✅ This file is encrypted and safe to keep locally
❌ NEVER upload secrets.enc to GitHub!


STEP 5: Update Dashboard with Your Key
-------------------------------------
1. Open: dashboard/dashboard.py
2. Find line 12: KEY = b'YOUR_ENCRYPTION_KEY_HERE'
3. Replace with your actual encryption key


STEP 6: Test the Dashboard
-------------------------------------
Run the dashboard:

python dashboard.py

You should see:
- 📦 Scripts tab
- 🤖 AI Generator tab
- 📊 Analytics tab
- ⚙️ Settings tab (showing API key status)

5. GITHUB PAGES DEPLOYMENT
================================================================================

OPTION 1: Automatic (Using GitHub Actions)
--------------------------------------------
1. Push your code to GitHub:

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/yourusername/yourrepo.git
git push -u origin main

2. The .github/workflows/deploy.yml will automatically deploy
3. Go to your repo Settings → Pages
4. Your site will be at: https://yourusername.github.io/yourrepo


OPTION 2: Manual Deployment
--------------------------------------------
1. Go to your GitHub repository
2. Settings → Pages
3. Source: Deploy from a branch
4. Branch: main / root
5. Save
6. Your site will be live in 1-2 minutes


Update Your data.json:
- The dashboard automatically updates data.json on GitHub
- Changes appear on your site within seconds

8. TROUBLESHOOTING
================================================================================

Problem: "secrets.enc not found"
Solution: Run encrypt_secrets.py first

Problem: "Failed to load secrets"
Solution: Check your encryption key in dashboard.py line 12

Problem: "GitHub Error: 401"
Solution: Check your GITHUB_TOKEN is valid

Problem: "AI generation failed"
Solution: 
- Check API key is configured
- Verify internet connection
- Check API key has credits/balance

Problem: Site not updating
Solution:
- Check GitHub Actions ran successfully
- Wait 1-2 minutes for GitHub Pages cache
- Hard refresh browser (Ctrl+F5)

Problem: Dashboard won't open
Solution:
- Install requirements: pip install -r requirements.txt
- Check Python version: python --version (need 3.8+)
- Check tkinter is installed
